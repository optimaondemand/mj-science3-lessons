Reading the Periodic Table
📖 TODAY'S FOCUS
The periodic table isn't just a list. It's an organized map, and once you know how to read it, it tells you a huge amount about an element before you ever touch it.
Elements aren't scattered randomly on the periodic table, they're grouped by shared properties. Elements in the same column tend to behave in similar ways, which means the table itself is a prediction tool.

📚 PART 1 · ROWS, COLUMNS, AND FAMILIES

Each box on the periodic table shows an element's symbol, name, and atomic number (the number of protons, which is what makes that element unique). Elements are arranged in rows and columns on purpose: elements in the same column (called a group or family) share similar chemical behavior, because they have similar arrangements of outer electrons. The noble gases (the far-right column) barely react with anything. The alkali metals (the far-left column) react violently with water. That's not a coincidence, it's the whole reason the table is organized this way.
